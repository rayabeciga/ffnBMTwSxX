# Python学生就业管理系统 python224

本项目是一款基于Python的学生就业管理系统，集成学院、专业、企业、岗位等多维度的信息管理功能，旨在为学校和学生提供一个便捷的就业信息交流平台。

## 技术栈

- 后端：Python、Django
- 前端：HTML、CSS、JavaScript、jQuery
- 数据库：MySQL
- 开发工具：PyCharm、Navicat
- 运行环境：Python 3.0以上、MySQL 8.0

## 功能模块

- 学院信息管理
- 专业信息管理
- 企业信息管理
- 岗位信息管理
- 管理员信息管理
- 学生信息管理
- 学生教育经历管理
- 学生项目经历管理
- 学生投递记录管理
- 修改个人信息与密码

## 系统角色

- 管理员
- 教师
- 学生

## 运行环境

- Python 3.0以上
- MySQL 8.0
